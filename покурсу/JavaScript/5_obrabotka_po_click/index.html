Типы событий JS

  • click - нажатие левой клавиши мыши
  • mouseover/mouseout - мыщи наводиться на (покидает) элемент
  • keydown/keyup - нажатие/отпускание клавиши на клавиатуре
  • focus - установк фокуса на элемент, например, input
  • submit - событие отправки формы
  • ...

Обработка события

  • Через атрибут тега - onevent (например, onclick, onmouseover, onsubmit
  • Вызов колбэка
  • Выборка ноды и добавление к ней события onevent (по ссылке)
  • Использование "лиснеров" addEventListener, removeEventListener

Доступ к содержимому ноды

  • Через контекст this
  • Через обьект события
